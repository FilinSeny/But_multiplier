# Флаги запуска

### `--m <M_WIDTH>`
Разрядность первого операнда (множимого).

- Обязательный параметр  
- Целое число строго больше 0  

### `--r <R_WIDTH>`
Разрядность второго операнда (множителя).

- Обязательный параметр  
- Целое число строго больше 0

### --out <FILENAME.v> 
Имя файла вывода
